# Source Projects

This folder is intended to contain all platform-specific and shared source projects for `PoePerfect Player`.

Current structure:

- [PoePerfect.Player.Windows](./PoePerfect.Player.Windows)
- [PoePerfect.Player.Web](./PoePerfect.Player.Web)
- [PoePerfect.Player.Core](./PoePerfect.Player.Core)
- [PoePerfect.Player.Android](./PoePerfect.Player.Android)
- [PoePerfect.Player.RaspberryPi](./PoePerfect.Player.RaspberryPi)

Current implementation status:

- Windows is the main desktop player.
- Web is an active React/Vite MVP and the planned base for future browser/webOS work.
- Android is an MVP.
- Core contains shared playlist/domain logic.
- Raspberry Pi/Linux is planned.
