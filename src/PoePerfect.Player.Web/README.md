# PoePerfect Player Web

React/Vite web MVP for browser and future webOS packaging.

## Run locally

```powershell
npm run dev
```

The local server defaults to:

```text
http://127.0.0.1:5173
```

The Vite dev server includes `/api/proxy?url=...` for catalog/API loading during browser testing. Video streams are played directly by the browser or TV media engine.
